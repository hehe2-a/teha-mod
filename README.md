# teha - ハックチェック (teha hack Check)

暇だから作ってみた

## 🛠 主な機能

- **自動チート検知**: modフォルダー内のmodをチェックして、チートクライアントなどを検知します。
- **警告画面 (Warning UI)**: チートクライアントが見つかった場合、警告画面を表示します。
